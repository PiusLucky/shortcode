--- title: auth middleware description: There are a few ways to apply the auth middleware in your project. tags: laravel auth date: 2019-09-15 --- // In the Controller constructor ... public function __constructor(){ $this->middleware('auth') // Apply login authorisation to all functions // or $this->middleware('auth')->only(['store', 'update']) // Apply login authorisation only to the given functions // or $this->middleware('auth')->except(['show']) // Apply login authorisation to all except the given functions } ... // In the routes file you can apply Auth Middleware to specific routes Route::get('item', 'ItemsController@index')->middleware('auth');
